# 12 — Applications

## Objectif

Regrouper les interfaces utilisateurs de GSIE. Les applications sont
des clients du moteur ; elles ne contiennent aucune logique métier.

## Responsabilités

- Décrire les interfaces prévues (Mobile, Desktop, Web, Plugins)
- Définir les responsabilités de chaque interface
- Garantir qu'aucune logique métier n'y réside

## Ce qui peut y être ajouté

- Descriptions d'interfaces
- Spécifications d'interaction
- Notes de design

## Ce qui est interdit

- Implémenter des écrans (Phase 1)
- Ajouter de la logique métier

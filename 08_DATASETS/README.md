# 08 — Datasets

## Objectif

Regrouper les jeux de données référencés par GSIE. Aucune donnée n'est
inventée : chaque dataset est documenté et sourcé.

## Responsabilités

- Cataloguer les datasets (climat, sol, flore, géographie, satellite)
- Documenter les métadonnées (source, licence, couverture, qualité)
- Gérer le versionnage des données

## Ce qui peut y être ajouté

- Fiches de métadonnées
- Catalogues de datasets
- Notes sur les licences

## Ce qui est interdit

- Intégrer des données sans métadonnées
- Ignorer les contraintes de licence (`19_LEGAL`)

- **07_KNOWLEDGE** : les données alimentent les connaissances

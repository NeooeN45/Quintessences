## Décision

Le Fondateur acte la création de l'**Encyclopédie de l'Écosystème** :
la plus grande base de données structurée, sourcée et traçable sur tout
ce qui touche à l'écosystème. Cette encyclopédie est **le produit
principal** de GSIE, pas un sous-produit des moteurs.

L'échelle visée est **million d'entrées minimum**, couvrant flore,
faune, sols, climat, hydrologie, pathologies, entomologie, mycologie,
interactions trophiques, dynamiques, sylviculture, biodiversité et
incendie.

## Contexte

- Le livrable 308 (Knowledge Base Seed) a amorcé la base avec 25
  connaissances manuelles.
- Le Fondateur exprime la vision d'une base à l'échelle du marché, pas
  d'un fichier markdown.
- Les livrables 301-307 définissent les méthodes, l'ontologie, le
  graphe, les datasets et le framework de preuve — ils restent valides
  comme fondation de l'Encyclopédie.
- L'implémentation technique (base graphe, pipelines NLP/LLM, API) se
  fait en Phase 4.

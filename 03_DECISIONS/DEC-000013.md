## Décision

Le Fondateur acte une restructuration majeure de l'écosystème
Quintessences :

1. **Myhunt → Artemis** (faune)
2. **GSIE-Ignis → Ignis** (incendies — l'application, pas le moteur)
3. **Hydro** (eau) et **Flora** (végétation) rejoignent l'écosystème
4. **QGISIA** reste comme plugin QGIS de Quintessences
5. **Unreal Engine** repositionné comme **Centre de Commandement GSIE**
6. **Réorganisation des dossiers** par application

## Architecture cible

```
Quintessences → GSIE (moteur) → GeoSylva, Artemis, Ignis, Hydro, Flora, QGISIA
```

GSIE est le système nerveux. Les applications sont expertes de leur
domaine. Le Centre de Commandement (UE 5.8) est le poste de pilotage
immersif où toutes les données convergent.

# 03 — Decisions

## Objectif

Tracer formellement chaque décision validée. Une décision est
exécutoire et impacte le projet.

## Responsabilités

- Être la source de vérité pour les décisions prises
- Lier chaque décision à son RFC et/ou Directive d'origine
- Permettre l'audit et la rétro-traçabilité

## Ce qui peut y être ajouté

- `DEC-XXXXXX.md` — nouvelles décisions

## Ce qui est interdit

- Prendre une décision sans la tracer ici
- Modifier une décision sans créer une nouvelle entrée

## Liens

- **01_DIRECTIVES** : origine des décisions
- **02_RFC** : débats ayant mené à la décision
- **22_PROJECT_MEMORY/DECISION_HISTORY** : chronologie consolidée

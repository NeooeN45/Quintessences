## Décision

Le Fondateur **étend l'écosystème Quintessences** avec deux nouvelles
spécialisations, portant le total à quatre applications clientes du
moteur GSIE.

## Architecture d'identité mise à jour

```
Quintessences (écosystème)
└── GSIE (moteur — General System Intelligence Engine)
    ├── GeoSylva       — app forestière (Android, inventaire/martelage)
    ├── Ignis     — spécialisation incendie (drones, simulation)
    ├── Artemis         — suivi cynégétique (Android + API + backoffice)
    └── QGISIA         — agent IA QGIS (plugin desktop, géospatial)
```

## Spécialisations

### GeoSylva (existante — DEC-000006)

Application Android professionnelle d'inventaire forestier, martelage
et dendrométrie — 100% hors-ligne. Première spécialisation de GSIE.

- **Repo GitHub** : `NeooeN45/GeoSylva`
- **Stack** : Kotlin, Jetpack Compose, Room, MapLibre
- **Statut** : Actif, branche `feature/pro-pricing-engine`

### Ignis (existante — DEC-000006, ex-GSIE-FEU)

Spécialisation incendie : surveillance par drones, simulation de
propagation, aide à la décision pour les COS.

- **Repo GitHub** : `NeooeN45/Quintessences` (architecture dans le dépôt moteur)
- **Stack** : Python, Rust, Go, TypeScript, C++/PX4
- **Statut** : Phase 2 (Architecture)

### Artemis (nouvelle)

Plateforme de suivi cynégétique premium orientée terrain. Monorepo
avec application Android, API NestJS et backoffice Next.js.

- **Repo GitHub** : `NeooeN45/Artemis`
- **Stack** : Kotlin/Compose, NestJS, Next.js, PostGIS, MinIO
- **Domaine** : gestion cynégétique, faune, territoires de chasse
- **Lien avec GSIE** : consommera à terme les moteurs GIS, Knowledge,
  Correlation et Learning pour l'analyse des populations, la prédiction
  de présence animale et la gestion durable des territoires.

### QGISIA (nouvelle, nom produit « GeoSylva AI »)

Agent IA intelligent pour QGIS. Plugin desktop qui route les demandes
en langage naturel vers le meilleur modèle, appelle les outils QGIS,
interroge le web et l'imagerie satellite, génère et exécute du PyQGIS.

- **Repo GitHub** : `NeooeN45/QGISIAPRO`
- **Stack** : Python, React 19, NVIDIA NIM, QGIS 3.16+/4.0
- **Domaine** : SIG desktop, agent IA géospatial, analyse multicouche
- **Lien avec GSIE** : QGISIA est l'**interface desktop** du moteur
  GSIE pour les professionnels SIG. Il consommera les moteurs GIS,
  Climate, Pedology, Botanical et Reasoning pour fournir des analyses
  environnementales expertes directement dans QGIS.

## Principes

1. **Chaque spécialisation est un repo GitHub séparé.** Aucune app
   n'est dans le dépôt du moteur (GSIE-CON-007, modularité).

2. **Chaque spécialisation est cliente du moteur GSIE.** Les apps
   consomment les 14 moteurs via des contrats d'interface
   (`04_ARCHITECTURE/ENGINE_INTERFACE_CONTRACTS.md`).

3. **Chaque README d'app référence Quintessences** comme écosystème
   parent et GSIE comme moteur sous-jacent.

4. **Le nom produit « GeoSylva AI » pour QGISIA** est un alias
   marketing. L'identifiant technique reste « QGISIA2 » (plugin QGIS).
   Aucun impact sur la gouvernance.

5. **Artemis et QGISIA restent autonomes** dans leur développement.
   L'intégration avec GSIE se fera progressivement (Phase 4 —
   Implémentation).

# DEC-000001 — GSIE est une Fondation scientifique

## Décision

Le projet GSIE est désormais officiellement une **Fondation
scientifique**. Le moteur GSIE est le **produit principal**. Les
applications (Mobile, Desktop, Web) ne sont que des interfaces.

## Contexte

Le projet a initialement été pensé autour de GeoSylva Mobile, une
application Android. Il est apparu que le véritable produit est le
moteur d'intelligence forestière, pas l'interface.

## Conséquences

- Aucune décision ne sera prise comme si Android était le produit
  principal.
- L'architecture est centrée sur les moteurs, pas sur les UI.
- La connaissance et la documentation priment sur le code.
- Chaque IA intervenant sur le projet devra respecter la Constitution.

## Décision

Le Fondateur **restructure l'identité du projet** en trois niveaux
hiérarchiques :

1. **Quintessences** — l'**écosystème**, marque umbrella qui regroupe
   toutes les spécialisations environnementales.
2. **GSIE** (General System Intelligence Engine) — le **moteur
   d'intelligence** modulaire, spécialisable par domaine environnemental.
   GSIE est le cœur technique de chaque spécialisation de Quintessences.
3. **GeoSylva** — l'**application forestière**, première spécialisation
   de GSIE. GeoSylva garde son nom historique et devient l'interface
   métier de la spécialisation forestière.

## Architecture d'identité

```
Quintessences (écosystème — marque umbrella)
└── GSIE (General System Intelligence Engine — moteur spécialisable)
    ├── GSIE Forestier → GeoSylva (app forestière)
    ├── GSIE-FEU (spécialisation incendie)
    └── [futures spécialisations : climat, eau, biodiversité, …]
```

## Changement d'acronyme

GSIE signifiait précédemment **GeoSylva Intelligence Engine**. L'acronyme
est redéfini :

- **Ancien** : GeoSylva Intelligence Engine (lié à la forêt uniquement)
- **Nouveau** : **General System Intelligence Engine** (moteur général,
  spécialisable par domaine)

GeoSylva n'est plus dans l'acronyme : c'est désormais une spécialisation
parmi d'autres, pas le projet entier.

## Conséquences

- **Quintessences** devient le nom public du projet (repo GitHub, marque,
  communication). Le dépôt GitHub `NeooeN45/Quintessences` est déjà en
  place.
- **GSIE** reste le nom technique du moteur. Toute la documentation
  existante (Constitution, RFC, Décisions, Architecture, moteurs) reste
  valable — GSIE est toujours le moteur, simplement généralisé à plusieurs
  domaines.
- **GeoSylva** est repositionné comme app forestière (une spécialisation
  de GSIE), au même titre que GSIE-FEU est une spécialisation incendie.
- Les 14 moteurs GSIE restent valables : ils sont la couche d'intelligence
  commune à toutes les spécialisations. Chaque spécialisation peut
  étendre les moteurs avec des modules domaine-spécifiques.
- La Constitution, les RFC, les Décisions et la gouvernance s'appliquent
  à l'écosystème Quintessences dans son ensemble. GSIE-CON-008 (Le Projet
  appartient à sa Vision) est respecté : la vision s'élargit à
  l'environnement global, la forêt en est le premier domaine.

## Ce qui ne change pas

- La Constitution (10 articles + sectorielles) reste valable.
- Les 14 moteurs GSIE restent le cœur technique.
- La gouvernance, la traçabilité, la hiérarchie documentaire.
- Le nom du dépôt GitHub (`Quintessences`).
- La licence propriétaire (All Rights Reserved).

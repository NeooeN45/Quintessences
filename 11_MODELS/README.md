# 11 — Models

## Objectif

Contenir les modèles — scientifiques, statistiques et d'apprentissage —
utilisés ou produits par GSIE. Un modèle est une représentation
formalisée d'un phénomène.

## Responsabilités

- Documenter les modèles (provenance, validation, métriques)
- Distinguer les modèles scientifiques des modèles d'apprentissage
- Assurer la traçabilité

## Ce qui peut y être ajouté

- Modèles de croissance, de production, de station
- Modèles pédologiques et hydrologiques
- Modèles d'apprentissage (poids, hyperparamètres)

## Ce qui est interdit

- Intégrer un modèle sans fiche de provenance
- Mélanger modèle et algorithme (`10_ALGORITHMS`)

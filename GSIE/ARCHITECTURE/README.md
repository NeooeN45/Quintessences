# 04 — Architecture

## Objectif

Décrire l'architecture logicielle, modulaire et scientifique de GSIE.

## Responsabilités

- Définir la structure des moteurs et leurs interactions
- Documenter les contrats d'interface
- Tracer les décisions d'architecture (ADR)

## Ce qui peut y être ajouté

- Schémas d'architecture
- Diagrammes de dépendances
- ADR (Architecture Decision Records)

## Ce qui est interdit

- Implémenter du code (Phase 1)
- Définir des règles métiers

# 17 — Documentation

## Objectif

Centraliser la documentation officielle (utilisateur, contributeur,
opérateur) de GSIE.

## Responsabilités

- Produire les guides d'utilisation et de contribution
- Maintenir les ADR et le changelog global
- Garantir que la documentation précède l'implémentation

## Ce qui peut y être ajouté

- Guides, ADR, changelogs
- Documentation de référence

## Ce qui est interdit

- Documenter des fonctionnalités non spécifiées

- **00_CONSTITUTION** : la documentation respecte la Constitution
- **05_SPECIFICATIONS** : source des documentations techniques

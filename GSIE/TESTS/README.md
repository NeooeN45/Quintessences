# 15 — Tests

## Objectif

Garantir la qualité, la reproductibilité et la non-régression de GSIE.

## Responsabilités

- Définir la stratégie de test (unitaire, intégration, non-régression)
- Valider la conformité scientifique des résultats
- Mesurer la couverture

## Ce qui peut y être ajouté

- Plans de test
- Stratégies de couverture
- Jeux de données de référence

## Ce qui est interdit

- Écrire des tests avant que les spécifications ne soient définies
  (Phase 1)

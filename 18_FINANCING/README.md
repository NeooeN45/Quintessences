# 18 — Financing

Documenter le modèle économique, les sources de financement, le budget par phase et la gouvernance financière de GSIE. Assurer la transparence et la traçabilité des flux financiers.

## Principes

- **La connaissance est un bien commun, l'infrastructure de service est payante.**
- Gratuit pour l'usage individuel (AGPL-3.0), payant pour l'usage institutionnel (licence commerciale).
- Diversification des sources : au moins 3 sources de financement actives simultanément jusqu'à l'autofinancement.
- Transparence : rapport financier annuel publié (CON-007).
- Autofinancement visé en Phase 6 (année 6).

## Responsabilités

- Assurer la transparence financière
- Diversifier les sources de financement
- Tracer les budgets par phase et par moteur
- Maintenir un runway ≥ 12 mois
- Publier le rapport financier annuel

## Ce qui est interdit

- Intégrer des données financières sensibles sans contrôle (`19_LEGAL`)
- Dépendre d'une seule source de financement
- Masquer un conflit d'intérêt
- Engager une dépense sans validation selon les seuils (cf. `FINANCIAL_GOVERNANCE.md`)

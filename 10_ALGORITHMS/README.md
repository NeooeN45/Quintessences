# 10 — Algorithms

## Objectif

Décrire les procédures computationnelles de GSIE, indépendamment des
moteurs qui les implémentent. Un algorithme est une méthode formelle ;
un moteur est son exécutable.

## Responsabilités

- Formaliser les algorithmes (pseudocode, complexité)
- Séparer la méthode de l'implémentation
- Garantir la reproductibilité

## Ce qui peut y être ajouté

- Pseudocode et descriptions formelles
- Analyses de complexité
- Références vers les modèles et moteurs

## Ce qui est interdit

- Implémenter du code exécutable (rôle de `09_ENGINES`)
- Définir un algorithme sans lien vers un modèle ou un moteur

## Liens

- **09_ENGINES** : implémente les algorithmes
- **11_MODELS** : modèles que les algorithmes résolvent

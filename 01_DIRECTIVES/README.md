# 01 — Directives

## Objectif

Regrouper toutes les Directives fondatrices du projet. Une Directive
formalise les décisions validées lors des discussions stratégiques entre
les fondateurs. Chaque discussion significative produit une nouvelle
Directive.

## Responsabilités

- Tracer les décisions stratégiques
- Définir le contexte, la vision et les engagements à un instant T
- Lister les documents et fichiers impactés
- Orienter les actions des IA

## Ce qui peut y être ajouté

- `ACTIVE/` : Directives en vigueur (ex. `GSIE-DIR-0001.md`)
- `ARCHIVED/` : Directives remplacées ou obsolètes
- Context Snapshots (toutes les 10 Directives)
- Master Context Snapshots (toutes les 100 Directives)

## Ce qui est interdit

- Modifier une Directive archivée
- Créer une Directive sans la structure requise (résumé exécutif,
  décisions, contexte, etc.)
- Contredire la Constitution

## Liens avec les autres dossiers

- **00_CONSTITUTION** : la Constitution prime sur toute Directive
- **03_DECISIONS** : une Directive produit des décisions
- **22_PROJECT_MEMORY** : mémoire des Directives

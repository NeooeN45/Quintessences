# 01/Directives — ARCHIVED

## Objectif

Conserver l'historique des Directives remplacées ou obsolètes.

## Responsabilités

- Préserver la traçabilité des évolutions stratégiques
- Permettre l'audit rétroactif

## Ce qui peut y être ajouté

- Directives déplacées depuis `../ACTIVE/` lorsqu'elles sont remplacées

## Ce qui est interdit

- Modifier le contenu d'une Directive archivée

## Liens

- Parent : `01_DIRECTIVES/`
- Sibling : `01_DIRECTIVES/ACTIVE/`

# 16 — Tools

## Objectif

Regrouper les utilitaires, scripts d'automatisation et chaînes de
construction de GSIE. Aucun outil ne contient de logique métier.

## Responsabilités

- Faciliter le développement, la validation et le déploiement
- Fournir des outils de scaffolding et de validation

## Ce qui peut y être ajouté

- Générateurs, validateurs, scripts CI/CD
- Outils de migration

## Ce qui est interdit

- Ajouter de la logique métier dans les outils

## Liens

- **04_ARCHITECTURE** : les outils respectent l'architecture
- **15_TESTS** : les outils peuvent automatiser les tests

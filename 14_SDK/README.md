# 14 — SDK

## Objectif

Fournir des bibliothèques clientes pour consommer GSIE depuis
différents langages et plateformes.

## Responsabilités

- Encapsuler l'API dans des bibliothèques ergonomiques
- Ne contenir aucune logique métier
- Couvrir Kotlin, Python, TypeScript

## Ce qui peut y être ajouté

- Définitions de SDK
- Exemples d'utilisation
- Notes de compatibilité

## Ce qui est interdit

- Implémenter des SDK (Phase 1)
- Ajouter de la logique métier

# GSIE Master Architecture

## Mission
Décrire l'architecture globale de GeoSylva Intelligence Engine (GSIE).

## Vision
GSIE est organisé en couches indépendantes :
1. Acquisition des données
2. Validation scientifique
3. Base de connaissances
4. Moteurs de raisonnement
5. Services métier
6. Interfaces (Mobile, Web, Desktop, API)

## Principes
- Une responsabilité par moteur.
- Communication par interfaces documentées.
- Toutes les décisions sont traçables.
- Les connaissances sont indépendantes du code.

## Objectif
Permettre l'évolution de GSIE pendant plusieurs décennies sans refonte complète.

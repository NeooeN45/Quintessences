# Architecture Principles

## Objectif
Garantir une architecture stable pendant plusieurs décennies.

## Principes

- Un moteur = une responsabilité.
- Couplage faible.
- Forte cohérence interne.
- APIs documentées.
- Dépendances minimales.
- Tests obligatoires.
- Traçabilité complète.
- Observabilité native.

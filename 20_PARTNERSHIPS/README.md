# 20 — Partnerships

## Objectif

Formaliser les partenariats scientifiques, institutionnels et
techniques de GSIE.

## Responsabilités

- Lister les partenaires et conventions
- Définir les contributions attendues et réalisées
- Faciliter les appels à collaboration

## Ce qui peut y être ajouté

- Liste des partenaires, conventions, accords

## Ce qui est interdit

- Ajouter un partenariat sans validation (`03_DECISIONS`)

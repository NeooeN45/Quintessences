# GSIE-CON-004 — Toute décision doit être explicable

Édition : Première Édition
Version : 1.0 (Draft)
Statut : À valider
Classification : Loi Fondamentale (Immuable)

## Déclaration
Toute recommandation produite par GSIE doit être entièrement explicable.

## Principe fondamental
Chaque résultat doit permettre de comprendre :
- pourquoi il est proposé ;
- quelles données sont utilisées ;
- quelles règles sont appliquées ;
- quel est son niveau de confiance ;
- quelles sont ses limites.

## Pourquoi cette loi existe
Une recommandation non expliquée ne peut être vérifiée, améliorée ni enseignée.

## Les cinq questions fondamentales
1. Pourquoi ?
2. Avec quelles données ?
3. Selon quelles règles ?
4. Avec quel niveau de confiance ?
5. Quelles limites ?

## Conséquences pour les moteurs
Chaque moteur devra produire :
- les sources utilisées ;
- les règles appliquées ;
- les calculs réalisés ;
- les hypothèses ;
- les incertitudes ;
- les moteurs sollicités.

## Cas d'application
Toute recommandation d'essence devra présenter les paramètres pédologiques, climatiques, topographiques, les publications scientifiques mobilisées et les facteurs limitants.

## Anti-Lois
Interdiction :
- de produire une boîte noire ;
- de masquer un calcul ;
- de masquer une incertitude ;
- d'afficher un score sans justification.

## Déclaration finale
« Une décision comprise est une décision qui peut être discutée, améliorée et transmise. »

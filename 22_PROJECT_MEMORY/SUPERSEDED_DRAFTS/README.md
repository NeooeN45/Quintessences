# Superseded Drafts

## Objectif

Conserver les brouillons de documents qui ont été supersédés par des
versions plus complètes ailleurs dans le projet. Le principe de GSIE est
**on archive, on ne supprime pas**.

## Règles

- Chaque brouillon comporte un en-tête indiquant où se trouve la
  version officielle
- Aucune modification du contenu d'origine (hormis l'en-tête)
- Ces documents ne sont pas la source de vérité — ils sont conservés
  pour la traçabilité historique

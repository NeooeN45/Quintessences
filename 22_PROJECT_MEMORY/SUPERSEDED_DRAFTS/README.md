# Superseded Drafts

## Objectif

Conserver les brouillons de documents qui ont été supersédés par des
versions plus complètes ailleurs dans le projet. Le principe de GSIE est
**on archive, on ne supprime pas**.

## Règles

- Chaque brouillon comporte un en-tête indiquant où se trouve la
  version officielle
- Aucune modification du contenu d'origine (hormis l'en-tête)
- Ces documents ne sont pas la source de vérité — ils sont conservés
  pour la traçabilité historique

## Propositions v5 non adoptées (2026-07-15)

Ces deux archives conservent intégralement le contenu de propositions
issues d'un brainstorming v5, **supersédées avant adoption** par la
convergence v6/v6.1. Elles sont conservées pour traçabilité historique
et n'ont **aucune valeur normative** : elles ne constituent ni une
décision active, ni une architecture adoptée.

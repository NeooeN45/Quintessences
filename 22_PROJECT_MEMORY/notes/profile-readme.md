# Neo

### Fondateur de Quintessences — écosystème d'intelligence environnementale

Je construis des **outils d'aide à la décision pour l'environnement** —
forêt, feux, faune, SIG — basés sur un moteur d'IA modulaire, traçable
et explicable. Le forestier reste le décideur ; l'IA assiste, jamais ne
décide à sa place.

## Projets phares

### Quintessences — moteur d'intelligence environnementale

[![Quintessences](https://img.shields.io/badge/Quintessences-GSIE%20Engine-blue?style=for-the-badge)](https://github.com/NeooeN45/Quintessences)

Moteur modulaire 14 moteurs (Evidence, Knowledge, Reasoning, Diagnostic,
GIS, Climate, etc.) avec gouvernance formelle, traçabilité scientifique
et architecture offline-first. Phase 2 : Architecture.

### GeoSylva — app forestière Android

[![GeoSylva](https://img.shields.io/badge/GeoSylva-Android-green?style=for-the-badge&logo=android&logoColor=white)](https://github.com/NeooeN45/GeoSylva)

Application professionnelle d'inventaire forestier, martelage et
dendrométrie — 100% hors-ligne. IBP CNPF, 12 couches cartographiques,
7 méthodes de cubage, GPS de précision, exports SIG.

### QGISIA — agent IA pour QGIS

[![QGISIA](https://img.shields.io/badge/QGISIA-Agent%20IA-orange?style=for-the-badge)](https://github.com/NeooeN45/QGISIAPRO)

Agent IA qui agit dans QGIS : langage naturel → routing modèle →
exécution PyQGIS, interrogation web et satellite. NVIDIA NIM, React 19.

### Myhunt — suivi cynégétique

[![Myhunt](https://img.shields.io/badge/Myhunt-Cynégétique-brown?style=for-the-badge)](https://github.com/NeooeN45/Myhunt)

Plateforme de suivi cynégétique premium — Android, API NestJS,
backoffice Next.js. Offline-first, PostGIS, Mapbox.

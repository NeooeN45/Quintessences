# GSIE Knowledge Method

Toute connaissance intégrée dans GSIE doit comporter :

- Identifiant unique
- Définition
- Domaine scientifique
- Source
- Niveau de preuve
- Version
- Historique
- Relations avec les autres connaissances
- Moteurs consommateurs
- Date de validation

Aucune connaissance ne peut exister uniquement dans le code.

# 07 — Knowledge

## Objectif

Contenir la base de connaissances structurée de GSIE : règles
expertes, ontologies, taxonomies, glossaires. C'est la mémoire formelle
du système expert.

## Responsabilités

- Structurer les connaissances forestières
- Garantir que chaque connaissance est sourcée et versionnée
- Servir de source unique de vérité pour les moteurs

## Ce qui peut y être ajouté

- Ontologies et taxonomies
- Règles expertes
- Glossaires et référentiels

## Ce qui est interdit

- Ajouter une connaissance sans source (`06_RESEARCH`)
- Mélanger connaissances et implémentation

## Liens

- **06_RESEARCH** : sources des connaissances
- **09_ENGINES** : les moteurs consomment les connaissances
- **08_DATASETS** : les données alimentent les connaissances
